# Python Learning - Setup and Introduction

## Overview
This module covers the essential setup and introduction to Python programming. It includes lessons on environment setup, running Python code, and basic concepts.

## Lessons

### Lesson 0.1: Environment Setup
- Installing Python
- Verifying the installation
- Setting up VS Code with Python extension
- Basic environment configuration

### Lesson 0.2: Running Python
- Using the Python REPL (interactive interpreter)
- Running Python scripts
- Understanding Python's execution model
- Basic command-line usage

## Learning Objectives
- Set up a Python development environment
- Understand different ways to run Python code
- Learn basic Python syntax and structure
- Get familiar with Python's interactive mode

## Key Differences from Java
- No need for explicit compilation
- Interactive REPL for quick testing
- Simpler environment setup
- More flexible execution options

## Prerequisites
- Basic computer literacy
- Access to a computer with internet connection
- No prior programming experience required

## Next Steps
After completing this module, proceed to Python Basics to learn about fundamental programming concepts.
